## ✨ 주요 기능
<b> 1. 실시간 소통(채팅)  </b> : Socket.io를 사용하여 실시간으로 소통이 가능하게 구현 <br> 
<b> 2. 랜덤이미지(겹치지않음) </b> : getRandomImageUrl( )을 사용하여 들어오는 접속자별로 이미지가 다르게 설정되게 구현 <br>
<b> 3. 현재접속자 확인 가능 </b> : 동시 접속되어 있는 사람들의 명단 확인 가능 <br>
<b> 4. 중복 닉네임 불가 </b> <br>
